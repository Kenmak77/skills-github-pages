How to install (Windows and MacOS)


1. Extract .zip file

2. Run Dolphin.exe/ProjectPlusfr.app and after the window Update pop up do 'Update'


3. Once finished. Add Brawl by default


If you have some trouble, use (Google Drive Mirror)


Note

MacOS could not authorize you to download the update

Run this command in Terminal to fix macOS issues:

sudo xattr -rd com.apple.quarantine path/to/ProjectPlusFR.app

Linux


Download P+FR_AutoUpdate.sh (Debian, Ubuntu, Mint, Arch)


Once installed, launch with P+FR.Desktop to keep the build update

P+FR.Desktop is in:

~/.local/share/P+FR/

Setting Dolphin Recommendation


Best Render and LET ANISOTROPIC FILTERING TO 1x !! (Otherwise PS2 doesn't look good)


For machines with powerful hardware, it's recommend using Exclusive Ubershaders alongside Compile Shaders Before Starting for a stutter-free experience.
